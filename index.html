About
CertName Input is a simple application that can be used to easily enter the name on the certificate without having to use any image editor software. You only need to prepare a blank certificate image template file with no name, and then use this app to input the name directly and customize font position and color.
How To Use
  1. Select a certificate image to be used. The recommended size is 3508 pixels x 2480 pixel, but you are free to use any size of certificate.
  2. Enter the full name of the recipient.
  3. Adjust the font size, color, and styling of the text.
  4. Adjust the text vertical or horizontal position.
  5. Click SAVE IMAGE.
FAQ

How to insert the certificate image?
To insert the certificate image, click on the Choose File and select the certificate image you wanna use.

How do I know the name of the font?
Well, first of all, you can use all the font that's installed on this device. But if you don't know what's the name of the font, the easiest steps are:

  1. Open the Notepad app.
  2. At the top menu bar, click on Format > Font...
  3. There you can test any font that's installed on your device. Remember the name of the font.
  4. Go back to CertName app, and enter the name of the font.

Why I can't see the recipient name?

  1. Make sure you already load a certificate image, to insert the image, click on the Choose File and select the certificate image you wanna use.
  2. After that, enter the recipient name. The text will automatically printed on the image.
  3. If you still couldn't see the image, try changing the vertical and horizontal slider position until you see the text. The smaller the image is, chances are you probably wanna set the slider to the left.
