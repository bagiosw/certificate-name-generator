To change the certificate, follow steps below:

  3. You will find a file named img.jpg, replace the file with yours. The file must be a *.jpg format.

About
CertName Input is a simple application that can be used to easily enter the name on the certificate without having to use any image editor software. You only need to prepare a blank certificate image file with no name, and then use this app to input the name directly and customize font position and color.
How To Use
  1. Select a certificate image to be used. The recommended size is 3508 pixels x 2480 pixel, but you are free to use any size of it.
  2. Enter the name of the recipient.
  3. Adjust the font size and color for the text.
  4. Adjust the position of the text vertically or horizontally.
